Molecule Visualizer

This is a molecule visualizer where you can upload .sdf files to a database and display them in a 3D environment. You can also add and delete elements to the database. You can reset the database aswell in case it is messed up. When displaying a molecule you can also see the numbers of atoms and bonds within said molcule. When displayed, you can also rotate the molecule on the x, y and z axis. The front end of this program is done through javascript/HTML/CSS and the back end was created with C library accessed through python with a SQL database.
